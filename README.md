# 💳 Hybrid Credit Risk Prediction System

A hybrid machine learning system that combines:

- **Model A:** Loan default prediction (XGBoost)  
- **Model B:** Behavioral risk scoring (Isolation Forest)  
- **Final Score:** Weighted combination of both  

---

## 🧠 Formula

FinalRisk = 0.7 × DefaultProbability + 0.3 × BehaviorRisk

---

## 🚀 Features

- Loan default prediction  
- Mobile money behavioral analysis  
- Hybrid risk scoring  
- Streamlit web interface
